Learning Objectives

Upon completion of this course, students will be able to:

  • Think algorithmically and break down complex problems into manageable steps
  • Code confidently in Python with solid understanding of programming fundamentals
  • Debug effectively and troubleshoot programming issues independently
  • Build complete projects from planning to presentation
  • Handle data using lists, dictionaries, and file operations
  • Apply problem-solving strategies to both programming and real-world challenges
  • Present technical work professionally with portfolio-ready projects

4-Week Course Structure

The course is divided into 4 intensive weeks, with 4 sessions per week building from visual programming to Python mastery:

🧠 Week 1: CS Foundations & Visual Programming

Sessions 1-4 | Introduction to Computer Science & Scratch

Sessions Included

  • Session 1: What is Computer Science? Career paths & opportunities
  • Session 2: Computational thinking & problem-solving fundamentals
  • Session 3: Introduction to Scratch - sprites, blocks, and basic logic
  • Session 4: Building your first interactive Scratch project

Skills You'll Master

  • Logical thinking
  • Problem decomposition
  • Visual programming
  • CS career awareness

What You'll Build

  • Create your first algorithm (daily routine flowchart)
  • Interactive storytelling game in Scratch
  • Animated character with movement controls

🐍 Week 2: Python Fundamentals

Sessions 5-8 | Real Programming Begins

Sessions Included

  • Session 5: Python basics - variables, data types, input/output
  • Session 6: Control structures - if/else statements and decision making
  • Session 7: Loops - for and while loops, repetition in programming
  • Session 8: Functions - organizing code and reusable components

Skills You'll Master

  • Python syntax
  • Variables & data types
  • Control flow
  • Functions

What You'll Build

  • Simple calculator with multiple operations
  • Number guessing game with user feedback
  • Basic quiz application with scoring

🐍 Week 3: Advanced Python & Data

Sessions 9-12 | Working with Data Structures

Sessions Included

  • Session 9: Lists - storing and manipulating collections of data
  • Session 10: Dictionaries - key-value pairs and data organization
  • Session 11: File handling - reading from and writing to files
  • Session 12: Putting it together - combining concepts for real applications

Skills You'll Master

  • Data structures
  • File operations
  • Data manipulation
  • Problem solving

What You'll Build

  • Todo list manager with file storage
  • Personal expense tracker with data analysis
  • Simple contact book with search functionality
  • Mini text-based adventure game

🚀 Week 4: Capstone Project Week

Sessions 13-16 | Build Your Showcase Project

Sessions Included

  • Session 13: Project planning & design - choosing and scoping your capstone
  • Session 14: Development session 1 - building core functionality
  • Session 15: Development session 2 - adding features and debugging
  • Session 16: Project presentation & portfolio showcase

Skills You'll Master

  • Project planning
  • Independent coding
  • Debugging
  • Presentation skills

Choose Your Capstone Project

  • 🏦 Personal finance manager with budgeting
  • 🎮 Text-based RPG game with character progression
  • 📊 Data analysis tool for a topic you care about
  • 🤖 Simple chatbot with personality
