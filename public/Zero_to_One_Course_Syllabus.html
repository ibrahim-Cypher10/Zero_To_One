Learning Objectives

Upon completion of this course, students will be able to:

  • Think algorithmically and break down complex problems into manageable steps
  • Code confidently in three programming languages: Scratch, Python, and C++
  • Debug effectively and troubleshoot programming issues independently
  • Build complete projects that demonstrate core computer science concepts
  • Succeed in university CS101 courses with confidence and reduced anxiety
  • Apply problem-solving strategies to both programming and real-world challenges
📋
